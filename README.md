# Project 6 — Causality & What-If Simulator

An interactive causal inference workbench. Define causal DAGs, estimate treatment effects using DoWhy, run counterfactual simulations, analyze policy interventions, and explore heterogeneous treatment effects — all across three real-world domains.

---

## The Core Idea

Most data science finds **correlations**. This project finds **causes**.

```
Standard ML:   "Ad spend and sales are correlated (r=0.74)"
               → Wrong conclusion: spending more in peak season inflates this

Causal ML:     "Ad spend causally increases sales by 0.008 units per dollar"
               → Correct: after removing the confounding effect of seasonality
```

The difference matters for decisions: a naive correlation might tell you to increase ad spend in peak season (where it has near-zero effect). The causal model tells you to spend in off-peak season (where ads actually work).

Three datasets with built-in confounders — designed so naive correlation gives the wrong answer:

## Core Concepts

### The Backdoor Criterion
```
A set Z satisfies the backdoor criterion if:
  1. Z blocks every path from Treatment → Outcome that has
     an arrow pointing INTO Treatment (confounding paths)
  2. Z contains no descendants of Treatment (mediators)

If Z exists: E[Y|do(T=t)] = Σ_z E[Y|T=t, Z=z] × P(Z=z)
```

### The Do-Operator
```
P(Y | T=t)       — conditioning (correlation)
                   "Among people I observed with T=t, what is Y?"
                   Biased by confounders

P(Y | do(T=t))   — intervention (causation)
                   "If I force T=t for everyone, what is Y?"
                   Unbiased — Pearl's do-calculus
```

### Structural Equations (HR example)
```python
# These equations define the data-generating process (causal DAG in code)
salary            = 30000 + years_at_company × 2500 + noise
work_life_balance = 3.0 - overtime × 1.2 + noise
job_satisfaction  = 2.0 + salary_effect + wlb × 0.4 + noise
attrition_logit   = 1.5 - wlb × 0.35 - jobsat × 0.40 - salary_effect + overtime × 0.60
# WLB coefficient (-0.35) is the TRUE causal effect we try to recover
```

### Refutation Tests
```
Placebo treatment test:
  Replace treatment with random noise
  → Real causal effect should disappear
  → If it doesn't, something is wrong

Random common cause test:
  Add a random variable as a new confounder
  → Estimate should be stable
  → Large change = model is fragile

Data subset test:
  Rerun on 80% of data
  → Estimate should be similar
  → Large change = overfitting to noise
```

## What-If vs Policy Simulation

| Feature | What-If | Policy Simulation |
|---------|---------|------------------|
| Unit | Individual | Population |
| Question | "What would happen to THIS person?" | "What would happen if we set T=t for EVERYONE?" |
| Use case | Personalized decision support | Policy design, budget allocation |
| Math | Individual counterfactual Y(t) | E[Y | do(T=t)] |

## Known Limitations

| Limitation | Impact | Fix |
|-----------|--------|-----|
| DAG must be manually specified | Wrong DAG = wrong estimates | Use PC algorithm for automated DAG discovery |
| Synthetic datasets only | True effects are known (cheating) | Connect to real data (IBM HR on Kaggle, WHO) |
| No confidence intervals displayed | Point estimates can be misleading | Add bootstrap resampling for CIs |
| Linear/logistic structural equations | Misses non-linear relationships | Use Gaussian processes or neural network SCMs |
| Sensitivity analysis is simplified | Not rigorous Rosenbaum bounds | Use `sensemakr` Python package |

**Q: What is Rosenbaum sensitivity analysis and what does a Gamma of 2.0 mean?**
Rosenbaum bounds quantify how sensitive a causal estimate is to hidden confounders. Gamma represents how much a hidden confounder could change the odds of receiving treatment. Gamma=1.0 means no hidden confounder. Gamma=2.0 means a hidden confounder that doubles the odds of treatment could be present — and the analysis asks: would the result still hold? If the result survives Gamma=3.0, it's robust. If it fails at Gamma=1.3, it's fragile and easily explained by a mild hidden confounder. This is mandatory reporting in credible observational studies.
